# PDF Unicode fontlari (Noto Sans)

Bu klasordeki TTF dosyalari PDF uretiminde gomulu olarak kullanilir.
Kullanicinin bilgisayarindaki fontlara bagimlilik yoktur.

Kaynak: Google Noto Sans (SIL Open Font License)

- NotoSans-Regular.ttf
- NotoSans-Bold.ttf

Tum PDF yardimcilari `src/utils/pdfFont.js` uzerinden bu fontlari yukler.
