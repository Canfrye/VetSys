# 🐾 VetSys - Veterinary Clinic Management System

VetSys is a modern veterinary clinic management system developed with **React** and **Vite**. The project aims to simplify daily clinic operations by providing an intuitive interface for managing customers, animals, appointments, vaccinations, examinations, stock, and reports.

## 🚀 Features

* 📋 Dashboard with statistics
* 👤 Customer Management
* 🐶 Animal Management
* 📅 Appointment Management
* 💉 Vaccination Tracking
* 🩺 Examination Records
* 📦 Stock Management
* 📊 Reports
* 🔍 Search & Filter
* ✏️ Edit and Delete Records
* 📄 PDF Report Generation
* 💾 LocalStorage Data Persistence

## 🛠 Technologies Used

* React
* Vite
* Material UI (MUI)
* JavaScript (ES6+)
* CSS
* LocalStorage

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/VetSys.git
```

Go to the project folder:

```bash
cd VetSys
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
